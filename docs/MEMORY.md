# Hiku Project Memory Index

## Project Overview
- **Client:** Hiku (skincare brand by Harper, age 15)
- **Agency:** Willa Creative
- **Platform:** Shopify
- **Domain:** hikubyharper.com
- **Contact:** Jessica Hou, VP Ecomm + Digital (jessica@hikubeauty.com)
- **Team size:** ~10
- **Target:** Gen Z, acne-prone/sensitive skin
- **Brand tone:** Credible but culturally fluent, editorial, approachable — "NYLON at its best"
- **Tagline:** "the coolest thing you can be is yourself"
- **Products:** 9 SKUs at launch ($22–$45), hero is Skincare Discovery Set ($49)
- **SKUs:** Guava Glow Moisturizer, Soft Matte Moisturizer, Oak Milk Toner, Green Juice Serum, Smooth Skin Face Scrub, Matcha Skin Cleanser, Rice Water Clay Mask, Glacé Kiss Balm (x2)
- **Experience Vision:** "The British Dollhouse of Curiosities" — rooms not pages, bento box content
- **Sales channels:** DTC, Ulta, TikTok Shop
