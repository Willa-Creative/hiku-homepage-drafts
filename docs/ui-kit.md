## Key Implementation Notes

1. **Two font families only:** Big Caslon (headings/display) + Helvetica Neue (body/UI)
2. **Dark/Light nav modes:** Both exist, likely context-dependent (dark on light pages, light on dark sections)
3. **Primary CTA is always Reddish Brown** with cream text, 4px border radius
4. **Product cards have 7+ variations** — final selection TBD during design review
5. **Footer is substantial** — newsletter capture with email + phone, editorial image, 4-column links
6. **Mobile scaling is proportional** — headings drop ~35% from desktop (56→36, 36→28, 24→20)
7. **Announcement banner** supports rotation (pause/arrows) for multiple messages
